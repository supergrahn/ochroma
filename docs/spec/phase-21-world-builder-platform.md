# Phase 21 — World Builder Platform

**Goal:** Ochroma becomes a platform where anyone can create, share, and monetise entire game worlds — not just assets, but complete game experiences built on the engine.

## 21.1 Visual Scripting (Ochroma Blueprints)
- Node-based visual scripting for game logic
- Drag-and-drop: events → conditions → actions
- No coding required for basic game mechanics
- Compiles to Wasm for mod distribution

## 21.2 World Marketplace
- Share complete game worlds (not just assets)
- Procedural worlds can be parameterised: "Generate a medieval city with 10k population"
- World templates with customisable parameters
- Revenue sharing for world creators

## 21.3 Live World Hosting
- Cloud-hosted persistent worlds
- Thousands of players in the same world
- World state persists across sessions
- Admin tools for world operators

## 21.4 Cross-World Travel
- Portal system connecting different worlds
- Carry items/reputation between worlds
- Unified identity across the Ochroma network
- Discovery: browse and visit other players' worlds

## 21.5 Creator Tools
- In-game terrain sculpting with physics preview
- Real-time Proc-GS rule editing with instant preview
- Material painting: brush spectral properties onto surfaces
- Cutscene editor with cinematic camera
- Audio placement with spatial preview

## Exit Criteria
- [ ] Visual scripting creates a working mini-game
- [ ] World marketplace hosts 10+ community worlds
- [ ] 100+ concurrent players in a persistent world
- [ ] Portal connects two different worlds
- [ ] Creator tools enable non-programmers to build content
